# ionic4-vue-app

http://i-doctor.sakura.ne.jp/dokuwiki/doku.php/ionic4/ionic-vue

## 開発環境

```
Windows10Pro 1803
nvm 1.1.7
node 10.15.1
npm 6.4.1
vue/cli 3.7.0

@ionic/vue 0.0.4
```
